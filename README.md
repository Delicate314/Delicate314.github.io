## 本地开发

如需在本地运行此网站：

1. 安装Ruby和Jekyll
2. 克隆此仓库
3. 运行 `bundle install` 安装依赖
4. 运行 `jekyll serve` 启动本地服务器
5. 访问 `http://localhost:4000` 查看网站
